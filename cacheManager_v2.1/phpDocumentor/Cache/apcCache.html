purgeIdentifierCache  [line 139]

  int purgeIdentifierCache( [string $identifier = '']  )

Will purge all caches that haves the same identifier

This can be of use on a multi-language website where the indexed content depends on the selected language. Normally you will only have the choice to delete all cache or one specific entry, with this little function you will delete all caches that have a certain identifier.

 Parameters:
string   $identifier:  Which cache we want to delete

API Tags:
Return:  The amount of caches deleted
